📍 Capa 1 — Prospección con Google Maps

🗺️ Manual Gratis

  • Buscar: "edificios de departamentos CABA"
  • Buscar: "administración consorcios"
  • Buscar: "inmobiliaria alquiler temporario"
  • Buscar: "guardamuebles Buenos Aires"
  • Cada ficha: nombre, teléfono, web, dirección
  • Desde web → sacar email de contacto

🔌 Google Places API ~$17/1000

  • Devuelve JSON: name, address, phone, website
  • Iterable por lat/lng + radio en km
  • Free tier: $200 USD/mes incluido
  • N8N HTTP Request → vuelca a Google Sheets
  • Tipos útiles: storage, real_estate_agency, moving_company

🛠️ Herramientas terceros

  • Outscraper.com — desde $4 USD, CSV con emails/teléfonos Pago
  • PhantomBuster — Maps + LinkedIn automatizado Pago
  • Hunter.io — dado un dominio, encuentra emails Freemium
  • Apollo.io — base de datos B2B completa Freemium

📊 Capa 3 — CRM y Seguimiento

📋 Opción A — Google Sheets Gratis · Recomendada para arrancar

Una hoja con estas columnas:

  • Nombre, Empresa, Email, Tel, Dirección
  • Fecha enviado, Abrió, Respondió
  • Estado: nuevo → contactado → respondió → reunión → cliente
  • Próxima acción + Notas
  • N8N lee y escribe automáticamente en la hoja

  • Tabla prospectos en la misma DB de TMC
  • Estados en enum: nuevo, contactado, reunion, cliente, descartado
  • Panel en admin TMC para gestionar el pipeline
  • Historial de emails enviados por prospecto
  • Notificaciones cuando alguien responde
  • Implementar cuando el volumen lo justifique

🔧 Notas técnicas para implementación

Evitar spam en emails masivos

  • Enviar máx. 50 emails/día desde Gmail gratuito
  • Usar dominio propio con SPF + DKIM configurado
  • Personalizar asunto y primer párrafo siempre
  • Incluir opción de darse de baja (legal)
  • Esperar 30–60 segundos entre envíos (N8N wait)

Integración con stack TMC

  • API endpoint POST /api/prospectos para recibir leads desde N8N
  • Webhook N8N para escuchar respuestas de Gmail
  • Tab "Prospectos" en panel admin TMC (fase 2)
  • Notificación push cuando alguien responde
  • Integración con Google Calendar para Zoom automático
