🔑 Decisiones de arquitectura clave
  • Escrow: el pago queda retenido hasta que el cliente confirma acceso (manual) o vencen 48hs sin confirmación (cron automático) → 85% al proveedor / 15% comisión TMC, registrado en movimientos_ledger en ambos caminos.
  • Reembolso por arrepentimiento: automático vía API de MercadoPago (PaymentRefund.total()) si el escrow todavía no se liberó al proveedor — agregado 23 jun 2026.
  • Chat restringido: habilitado desde confirmada hasta que se libera el depósito en garantía (escrow_liberado = 1), cierre simétrico para cliente y proveedor.
  • Consultas públicas: consultas_espacio tiene collation distinta a espacios — nunca JOIN directo, siempre dos queries separadas y merge en JS.
  • Vigencia de publicaciones: 90 días corridos desde la creación, con aviso 15 días antes del vencimiento.
  • Filtro de contenido: regex puro (no IA), 4 categorías, aplicado en frontend (UX) y backend (seguridad).
