# Security Policy — TodasMisCosas.com

## Reporting a Vulnerability

Si encontrás una vulnerabilidad de seguridad en TodasMisCosas.com, por favor repórtala de forma responsable **antes** de divulgarla públicamente.

**Email:** seguridad@todasmiscosas.com  
**Alternativa:** alejandro.laporte@gmail.com

Incluí en tu reporte:
- Descripción del problema y su impacto potencial
- Pasos para reproducirlo
- Versión o commit afectado (si podés determinarlo)

Nos comprometemos a responder dentro de las 48 horas hábiles y a coordinar un fix antes de cualquier divulgación pública.

## Scope

Esta política aplica a:
- `todasmiscosas.com` y subdominios
- API en `todasmiscosas.com/api`
- Repositorio `github.com/alex2580/prueba`

## Out of scope

- Ataques de denegación de servicio (DoS/DDoS)
- Spam o abuso del formulario de contacto
- Vulnerabilidades en dependencias de terceros no explotables en este contexto
